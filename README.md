# StationCore Home

## システム概要
StationCore全体のポータル（入口）として、各業務アプリへのアクセス、通知、業務状況の確認、共通設定を提供するホームアプリです。

## 基本コンセプト
* 入力は一度だけ
* データは一元管理
* 各アプリは独立（マイクロフロントエンド思想）
* スマホ利用を最優先（モバイルファースト）

## Homeが保持するデータ
業務データは保持せず、集約・表示に徹します。保持するのは以下の個人設定のみです。
* お気に入りアプリ
* 最近使用したアプリ
* ホーム配置
* テーマ（ダークモード等）
* 通知設定

## 技術スタック（予定）
* フロントエンド: React (Vite)
* バックエンド/BaaS: Firebase (Auth, Firestore)
* UIフレームワーク: Tailwind CSS / shadcn/ui 等
